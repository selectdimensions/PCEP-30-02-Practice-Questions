Instructions for Running the Python Practice Questions

Step 1: Prepare Your Files

Ensure all your files (HTML, JavaScript, and JSON files) are in the same directory. For example, they might be in a folder called "Python Practice Questions" on your desktop.

Step 2: Set Up the Express.js Server

Before running the application, you need to set up and start the Express.js server:

  1. Make sure you have Node.js installed on your computer.
  2. Open a terminal or command prompt and navigate to your project directory.
  3. Run the following command to install the necessary dependencies: 
    npm install express cors
  4. Create a file named server.js in your project directory and copy the Express.js server code into it.

Step 3: Start the Express.js Server

In your terminal or command prompt, run the following command:

node server.js

You should see a message like "Server running at http://localhost:3000"

Step 4: Open the Practice Questions in Your Web Browser

Open your web browser and go to:

http://localhost:3000/practice_test.html

Step 5: Using the Practice Questions

You should now see the practice questions in your browser. Use them as needed for your study. Your results will be saved automatically.

Step 6: Viewing Saved Results

To view your saved results, go to:

http://localhost:3000/results.html

Step 7: Stopping the Server

When you're done:

  • Go back to your terminal or command prompt window
  • Press Ctrl+C to stop the server
  • You can close the terminal or command prompt window

Modifying Questions (for Developers)

If you need to modify the questions:

  1. Open the questions.json file in a text editor
  2. Modify the questions, answers, and explanations as needed
  3. Save the file
  4. Restart the server for the changes to take effect

Note on File Paths

The application now uses relative file paths, so as long as all files are in the same directory, you shouldn't need to modify any paths in the code.

Troubleshooting

If you encounter any issues:

  • Make sure all files are in the same directory
  • Check that the server is running and you're accessing the correct URL
  • Check the browser's developer console for any error messages
  • Ensure you have the necessary permissions to read/write files in the project directory

For the best experience, we recommend using the Express.js server method described above.
